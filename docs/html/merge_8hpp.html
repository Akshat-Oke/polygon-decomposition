Find the next vertex in all polygons.

LP(vj) is a list containing pairs (k, vr) where k is the index of a polygon containing vj as one of its vertices and vr is the next vertex to vj in that polygon k, but these pairs are in LPvj only if vr is not the consecutive vertex of vj in the initial polygon, i.e., if vjvr is a diagonal,
